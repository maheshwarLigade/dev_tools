# DevTools - The Ultimate Developer Utility Belt

DevTools is a powerful, local-first web application designed to simplify the daily tasks of developers. It offers a comprehensive suite of utilities including formatters, converters, encoders, and debuggers—all within a beautiful, cohesive interface.

![DevTools Dashboard](https://raw.githubusercontent.com/devtools/branding/main/screenshots/dashboard.png)

## ✨ Key Features

- **Formatters:** JSON, XML, SQL, HTML, CSS, JavaScript, PHP.
- **Converters:** JSON to YAML/CSV/PHP, cURL to Code, HTML to JSX, SVG to CSS, Base Converter.
- **Encoders & Decoders:** Base64, URL, JWT, HTML Entities, Hex/ASCII, Cert Decoder.
- **Generators:** Secure Hash (SHA, MD5), UUID (v4), Lorem Ipsum, Password Generator.
- **Inspectors & Testers:** Regex Tester with capture group highlighting, Cron Parser, URL Parser, String Inspector, Text Diff Checker.
- **Previews:** HTML/CSS Preview, Markdown Preview.
- **Color Tools:** HEX/RGB/HSL Converter with tonal analysis.

## 🚀 Desktop Application

DevTools can be built as a standalone desktop application for Windows, macOS, and Linux using Electron.

### Building for your OS

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/devtools.git
   cd devtools
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Build executable:**
   ```bash
   # Build for current OS
   npm run build:desktop
   ```

### Download Executables
You can find the latest pre-built binaries for all platforms in our [GitHub Releases](https://github.com/your-username/devtools/releases).

## 🛠️ Tech Stack

- **Framework:** React 19 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS 4
- **Animations:** Motion
- **Icons:** Lucide React
- **Validation:** Ajv (JSON Schema)
